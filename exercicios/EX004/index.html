Itálico/Ênfase

Nesta frase temos um termo em itálico usando a tag i para não semântica.

Nesta frase vamos usar a tag em, consequentemente o termo ficará em itálico mas com sentido semêntico.

Texto marcado

Podemos criar um texto marcado usando a tag mark.

Texto inserido e deletado

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois com a tag ins.

Também podemos marcar um texto como excluído para indicar que ele deve ser lido mas não considerado com a tag del.

Texto subscrito e sobrescrito

Para inserir coisas elevadas ou do tipo usamos o sup como tag ou para coisas sobre ou do tipo usamos o sub como tag:
